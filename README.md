Mata Kuliah **Algoritma dan Pemrograman Lanjutan**  
Program Studi **Pendidikan Teknologi Informasi**  
Universitas Muhammadiyah Tasikmalaya

---

## Deskripsi Program

Program ini merupakan program berbasis console yang dibuat menggunakan bahasa pemrograman **C**.  
Program digunakan untuk membantu mengelola data nilai mahasiswa dalam satu kelas, mulai dari
menyimpan data, menampilkan data, mengubah dan menghapus data, hingga melakukan perhitungan nilai.

Program ini dibuat untuk melatih penggunaan:
- array satu dimensi,
- array dua dimensi (matriks),
- perulangan,
- serta fungsi rekursif.

---

## Fitur Program

Program menyediakan menu dengan beberapa pilihan, antara lain:
1. Menampilkan seluruh data mahasiswa
2. Menambahkan data mahasiswa
3. Mengubah nilai mahasiswa berdasarkan NIM
4. Menghapus data mahasiswa
5. Menghitung hasil pengolahan nilai
6. Menjalankan fungsi rekursif (faktorial dan Fibonacci)
7. Keluar dari program
